Installation¶

[1]

This part of the documentation covers the installation of SimpleTR64. The first step to using any software package is getting it properly installed.

Distribute & Pip¶

Installing SimpleTR64 is simple with pip, just run this in your terminal:

$ pip install simpleTR64

or, with easy_install:

$ easy_install simpleTR64

But, you really shouldn’t do that.

Get the Code¶

SimpleTR64 is actively developed on GitHub, where the code is always available.

You can either clone the public repository:

$ git clone git://github.com/bpannier/simpletr64.git

Download the tarball:

$ curl -OL https://github.com/bpannier/simpletr64/tarball/master

Or, download the zipball:

$ curl -OL https://github.com/bpannier/simpletr64/zipball/master

Once you have a copy of the source, you can embed it in your Python package, or install it into your site-packages easily:

$ python setup.py install
[1]Thanks for the template to http://docs.python-requests.org/en/latest/user/install/
